
# Converter

Converter.py converts various units, including weight, length, temperature, and more. This was my first Python project so I overcomplicated it when it could have been a lot simpler.


## Features

- Converts temperatures
- Converts weights
- Converts lengths

## Requirements

- Python 3.x
- Linux or VSCode's Python Terminal

## Running the file

1. Download Converter.py
2. Run **chmod +x /path/to/Converter.py** in your linux terminal
3. Now use **./path/to/Converter.py** to open it
